# OBR_2024

Código e projeto do robô do SESI-435 de Votuporanga

## Coisas a fazer

- [ ] Sensores do cruzamento
- [ ] Curva de 90 graus
- [ ] Deteção do verde
- [ ] Algoritmo do cruzamento (reencontrando a linha)
- [ ] Aperfeiçoar segue linha
- [ ] Sensor ultrassônico
- [ ] Desviar da caixa
